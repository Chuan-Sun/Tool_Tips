## man
`man man`

## ssh
secure shell
例：`ssh -p 50022 my@127.0.0.1`
`-p` 后面是端口
`my` 是服务器用户名
`127.0.0.1` 是服务器 IP

## ls
list files
`ls`、`ll`、`la`
![[la结果.png]]
file type：
`d`：目录
`-`：文件；
`l`：链接文档(link file)；
`b`：装置文件里面的可供储存的接口设备(可随机存取装置)；
`c`：装置文件里面的串行端口设备，例如键盘、鼠标(一次性读取装置)。
